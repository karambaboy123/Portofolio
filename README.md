# Persoonlijke Groei Tracker

Een interactieve single-page applicatie voor persoonlijke groei — gebouwd met HTML, CSS en vanilla JavaScript.

## Functies

- **Nulmeting instellen** — selecteer leerdoelen en stel je startscores in (0–100)
- **Spiderplot (radardiagram)** — live visueel overzicht van al je leerdoelen
- **Statbars (FIFA-stijl)** — horizontale voortgangsbalken met delta-badges
- **Bestanden uploaden & analyseren** — via de Anthropic API (claude-sonnet-4-6)
- **Delta-badges** — klikbaar, opent modal met de volledige AI-uitleg
- **Wit/blauw design** — Trello-achtig, clean en responsive

## Gebruik

1. Open `index.html` direct in een browser — geen installatie nodig
2. Selecteer je leerdoelen en stel je nulmeting in
3. Voer je [Anthropic API key](https://console.anthropic.com/) in
4. Upload tekstbestanden (.txt, .md, .csv, .json) om je groei te laten analyseren

## Technisch

| Onderdeel | Technologie |
|---|---|
| UI | Vanilla HTML/CSS/JS |
| Radardiagram | Chart.js 4 (CDN) |
| Lettertype | Inter (Google Fonts, CDN) |
| AI-analyse | Anthropic API (`claude-sonnet-4-6`) |
| Installatie | Geen — één enkel HTML-bestand |
