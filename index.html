Persoonlijke Groei Tracker

Stel je nulmeting in en start je groeitraject

Installeer Ollama op ollama.com, daarna run je in een terminal:
ollama serve  en  ollama pull llama3.2

Open LM Studio (lmstudio.ai), laad een model en start de Local Server via het server-icoon.
De app verbindt automatisch met http://localhost:1234.

Gratis key aanmaken op aistudio.google.com/apikey — geen betaling vereist.
